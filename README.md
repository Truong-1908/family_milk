# Family Milk Project
Dự án truy xuất nguồn gốc sữa.

## Hướng dẫn chạy (Mới nhất):

**Cách 1 (Dễ nhất - Khuyên dùng):**
- Chỉ cần chạy file `START.bat` ở thư mục này. Nó sẽ tự động bật cả Backend và Frontend cho bạn.

**Cách 2 (Thủ công):**
1. Mở terminal tại thư mục `backend`, chạy: `npm start`
2. Mở terminal khác tại thư mục `frontend`, chạy: `npm run dev`

Link truy cập: http://localhost:5173
